CI/CD Pipeline for a Containerized Web App

Network Topology Design & Automation
Network admin tasks (log parsing, file transfers) were manual and environment-dependent. Needed portability.
Designed and simulated enterprise-grade topologies in Cisco Packet Tracer — VLANs, subnetting, RIP/OSPF routing — and validated with traceroute on Linux. Then automated all admin tasks with Bash scripts and cron jobs, containerized the whole setup in Docker for consistent execution anywhere.
